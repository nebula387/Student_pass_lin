# Exam Performance Analysis & Prediction

В рамках проекта проведён полный цикл анализа данных: определены цели и метрики (accuracy, F1-score, ROC-AUC), загружены и очищены данные, выполнена предобработка и исследовательский анализ (EDA), проверены гипотезы о влиянии признаков на сдачу экзамена. Построена и обучена модель логистической регрессии, оценены её результаты и выявлены ключевые факторы успеха студентов. На основе анализа подготовлены практические рекомендации для улучшения академической успеваемости и раннего выявления студентов группы риска.

Иточник данных датасет Kaggle: (https://www.kaggle.com/datasets/ishanjha100/student-passfail-data)
