# Student_pass_lin

### Предсказание сдачи экзамена студентами на базе Sklearn

Использована библитека с Kaggle https://www.kaggle.com/datasets/ishanjha100/student-passfail-data

содержащая даные:
- id студента
- Процент посещаемости
- Оценка за домашнее задание
- Результаты промежуточного экзамена
- Часы, потраченные на учебу
- Зачет или незачет

все данные количественные (кроме id), без пропусков, последняя колонка -  бинарная целевая переменная
